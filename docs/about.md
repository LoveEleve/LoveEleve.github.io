# 关于本站 📖

## 站点介绍

这是一个基于 Docsify 搭建的个人文档站点，用于记录和分享学习心得。

## 为什么创建这个站点？

- 📚 **知识沉淀**：系统化整理学习内容
- 🧠 **加深理解**：通过写作巩固知识
- 🤝 **分享交流**：帮助他人，共同进步
- 📝 **方便查阅**：随时随地访问笔记

## 技术栈

- **框架**：Docsify
- **托管**：GitHub Pages
- **主题**：Vue Theme + Dark Mode
